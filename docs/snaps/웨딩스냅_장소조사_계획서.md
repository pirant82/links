# 🌿 서울/근교 웨딩 스냅 '딥 그린(Deep Green)' 장소 조사 계획서

## 1. 개요
* **목표**: 골프장 같은 단순 잔디밭이 아닌, 나무가 울창하고 싱그러운 '숲' 느낌의 웨딩 스냅 장소 발굴.
* **핵심 키워드**: #울창한나무 #싱그러움 #포레스트 #프라이빗 #수직적질감

## 2. 장소 선별 기준 (Filtering Logic)
1. **Verticality (수직성)**: 키 큰 나무들이 배경을 꽉 채워 깊이감을 주는가? (메타세쿼이아, 자작나무 등)
2. **Density (밀도)**: 배경에 인공물이 보이지 않고 자연에만 집중할 수 있는가?
3. **Variety (다양성)**: 이끼, 연못, 덩굴 등 다채로운 식생이 어우러져 있는가?
4. **Practicality (실용성)**: 웨딩 드레스를 입고 이동이 가능한가? 촬영 허가가 용이한가?

## 3. 조사 대상 카테고리
| 카테고리 | 특징 | 대표 후보지(예정) |
| :--- | :--- | :--- |
| **메타세쿼이아/가로수길** | 웅장하고 직선적인 미, 이국적 분위기 | 서울숲, 양재 시민의숲, 하늘공원 |
| **야생 숲/공원** | 정제되지 않은 자연스러움, 비밀 정원 느낌 | 올림픽공원 숲길, 인천대공원 |
| **온실/보타닉 가든** | 계절 무관, 밀도 높은 초록빛 | 창경궁 대온실, 서울식물원 |
| **프라이빗 베뉴** | 유료 촬영 위주, 정돈된 고급스러움 | 삼청각, 부천 상동호수공원 등 |

각 후보지는 아래 형식으로 상세히 분석합니다.
* **장소명**: (예: 서울숲 거울연못 인근)
* **분위기(Vibe)**: (예: 요정의 숲, 몽환적인 햇살)
* **주요 수종**: (예: 메타세쿼이아, 자작나무)
* **촬영 팁**: (예: 오전 10시 이전 빛이 부드러움)
* **주의사항**: (예: 사전 촬영 허가증 필수, 상업 촬영비 발생 등)
